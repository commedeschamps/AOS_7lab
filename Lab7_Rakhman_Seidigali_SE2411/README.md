## How to Run

From inside this folder, run:

```bash
python code/deadlock_detection.py
python code/bankers_algorithm.py
python code/thread_deadlock.py
python code/deadlock_prevention.py
python code/deadlock_recovery.py
```

If your system uses `python3`, use:

```bash
python3 code/deadlock_detection.py
python3 code/bankers_algorithm.py
python3 code/thread_deadlock.py
python3 code/deadlock_prevention.py
python3 code/deadlock_recovery.py
```

For Windows Task 5 observation only, the helper script can be run with:

```bash
python code/deadlock_recovery_observe.py
```
